# TEXHEC
## What is TEXHEC ?
TEXHEC is a game during development.

In TEXHEC we can win by:
- technological win
- total enemies annihilation

Point of the game is to force a player to find a way to manage seemingly endless world.
Where player has to find middleground between automating actions of dozens of units and performing actions manually.
